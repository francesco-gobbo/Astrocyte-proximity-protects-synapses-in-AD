# Astrocyte-proximity-protects-synapses-in-AD
Analysis for data in Gobbo et al. EJN 2026

Raw image analysis was performed in R and data processing was done in Python and R (statistics)

Python files:

Bleaching analysis: Bleaching_Analysis.ipynb

Single calcium event calculation: Single_event_calculation.ipynb

Event extraction: Synapse_Event_Detection_Runme.ipynb

R files:

Files are cointained in R Analysis folder in a single R project

Data can be retreieved from University of Edinburgh DataStore (see Gobbo et al. 2026 for link)
